# Query Archive

This section preserves natural-language queries made to the wiki across sessions.

## How Queries Work

Queries are prompts submitted by researchers working with this knowledge base. Each query represents a specific information need or research question.

## Format

Each query is stored as a markdown file with:
- The original question or prompt
- Relevant context from the wiki
- Any answers or generated content
