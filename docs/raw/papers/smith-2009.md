# Correspondence of the brain's functional architecture during activation and rest

**Authors:** Stephen M Smith et al.  
**Year:** 2009  
**Venue:** PNAS 106(31):13040-13045

## Summary
Demonstrated strong correspondence between task-evoked activation patterns and resting-state functional connectivity patterns. Provided evidence that intrinsic connectivity networks reflect the brain's functional organization for task execution.

## Key Contributions
- Task activation maps correlate with resting-state networks
- Validates resting-state as revealing functional organization
- Supports use of resting-state for understanding cognitive architecture

## Significance for Whole-Brain Modeling
Supports the approach of using resting-state data to parameterize models that can then simulate task states. The correspondence suggests shared underlying mechanisms.
