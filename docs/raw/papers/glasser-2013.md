# The Minimal Preprocessing Pipelines for the Human Connectome Project

**Authors:** Matthew F. Glasser, Steven M. Smith, et al.  
**Year:** 2013  
**Venue:** NeuroImage

## Key Contributions

- Standardized preprocessing pipelines for HCP multimodal data
- Surface-based analysis framework for cortical mapping
- Alignment of structural and functional data in a common space
- Open-source implementation (HCP Pipelines)

## Abstract Summary

The HCP minimal preprocessing pipelines provide standardized processing for structural MRI, functional MRI, and diffusion MRI data. The pipelines emphasize surface-based analysis, cross-modal alignment, and minimal yet effective preprocessing to preserve data quality while enabling comparison across subjects.

## Significance

The HCP preprocessing pipelines have become a standard for neuroimaging data processing. The surface-based approach and cross-modal alignment have influenced how the field processes multimodal neuroimaging data.
