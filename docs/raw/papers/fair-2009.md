# Functional Brain Networks Develop from a "Local to Distributed" Organization

**Authors:** Damien A. Fair, Nico U.F. Dosenbach, et al.  
**Year:** 2009  
**Venue:** PLoS Computational Biology

## Key Contributions

- Demonstrated developmental shift from short-range to long-range connectivity
- Showed that functional networks become more distributed with age
- Identified specific network changes in default mode and attention networks
- Linked connectivity changes to behavioral improvements

## Abstract Summary

Using resting-state fMRI in 210 participants aged 7-31, the authors found that functional brain networks transition from locally organized configurations in children to distributed patterns in adults. Short-range connections decrease while long-range connections strengthen, particularly in the default mode network and fronto-parietal attention networks.

## Significance

Provides evidence for how functional connectivity reorganizes during development, with implications for understanding cognitive maturation and developmental disorders.
