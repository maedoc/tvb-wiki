## Summary
Documentation for AUTO-07P, the standard software for continuation and bifurcation analysis of ODEs. Used extensively in computational neuroscience for exploring neural model parameter spaces and detecting bifurcations.

## Key Concepts
- Pseudo-arclength continuation
- Bifurcation detection algorithms
- Branch switching techniques
- Homotopy methods
- Boundary value problem continuation

## Relevance to TVB
- Primary tool for [[bifurcation analysis]] in neural models
- Parameter continuation for [[neural mass model]] exploration
- Used in [[epilepsy modeling]] for seizure threshold analysis
