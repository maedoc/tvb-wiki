# Effects of Aging on Functional Connectivity of the Default Mode Network

**Authors:** Jessica R. Andrews-Hanna, Andrei H. Snyder, et al.  
**Year:** 2007 (published 2008)  
**Venue:** Neurobiology of Aging

## Key Contributions

- First study to systematically examine DMN connectivity changes in aging
- Found reduced connectivity within DMN in older adults
- Showed correlation between DMN connectivity and cognitive performance
- Demonstrated specific vulnerability of DMN to aging

## Abstract Summary

Using resting-state fMRI, the authors found that functional connectivity within the default mode network decreases with age. Older adults show reduced correlation between posterior cingulate cortex and medial prefrontal cortex, and this reduction correlates with cognitive decline.

## Significance

Landmark study establishing DMN connectivity as a marker of brain aging, with implications for understanding cognitive decline and dementia risk.
