# Memory and Executive Function in Aging and AD

**Author:** Randy L. Buckner  
**Year:** 2004  
**Venue:** Neuron

## Key Contributions

- Proposed cognitive reserve as capacity to withstand brain pathology
- Distinguished between brain reserve (structural) and cognitive reserve (functional)
- Discussed compensation mechanisms in aging brains
- Linked reserve concepts to neuroimaging findings

## Abstract Summary

This review discusses how memory and executive function decline with age and Alzheimer's disease, focusing on the concepts of brain reserve and cognitive reserve. Reserve factors allow some individuals to maintain function despite accumulating pathology, through both passive (brain reserve) and active (compensation) mechanisms.

## Significance

Seminal paper introducing reserve concepts to cognitive neuroscience, foundational for understanding individual differences in aging trajectories.
