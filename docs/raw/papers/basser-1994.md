# MR diffusion tensor spectroscopy and imaging

**Authors:** Peter J Basser, James Mattiello, Denis LeBihan  
**Year:** 1994  
**Venue:** Biophysical Journal 66(1):259-267

## Summary
Introduced diffusion tensor imaging (DTI) as a method for characterizing water diffusion anisotropy in biological tissues. Established the mathematical framework relating diffusion measurements to tissue microstructure.

## Key Contributions
- Diffusion tensor mathematical model
- Relationship between diffusion and tissue structure
- Foundation for DTI tractography
- Non-invasive white matter characterization

## Significance for Whole-Brain Modeling
Provides the structural connectivity data that constrains whole-brain network models. DTI-derived structural connectivity is a primary input to TVB and similar platforms.
