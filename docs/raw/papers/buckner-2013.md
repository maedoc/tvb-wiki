# The organization of the human cerebellum estimated by intrinsic functional connectivity

**Authors:** Randy L Buckner et al.  
**Year:** 2013  
**Venue:** Neuron 80(5):1189-1207

## Summary
Large-scale analysis of intrinsic functional connectivity revealing organizational principles across the cerebral cortex and cerebellum. Demonstrated parallel processing streams and network architecture.

## Key Contributions
- Large-scale resting-state network organization
- Cerebellar functional connectivity patterns
- Hierarchical network architecture

## Significance for Whole-Brain Modeling
Provides empirical targets for large-scale network models. The identified network organization constrains how models should be structured to capture brain-wide dynamics.
