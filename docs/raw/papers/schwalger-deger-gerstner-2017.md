## Summary
Statistical theory linking microscopic spiking dynamics to macroscopic population activity in cortical columns. Derives population equations from single-neuron properties.

## Key Concepts
- Population density methods
- Fokker-Planck approach
- Mean-field approximations
- Cortical column dynamics
- Quasi-renewal approximation
