# Bifurcation analysis of Jansen's neural mass model

**Authors**: Jonathan Touboul, Fabien Wendling, Bruno Bellanger, Patrick Chauvel, Olivier Faugeras  
**Published**: 2011  
**Journal**: Neural Computation  
**DOI**: 10.1162/NECO_a_00151

## Summary

Comprehensive bifurcation analysis of the Jansen-Rit model, mapping parameter space to dynamical regimes including fixed points, limit cycles, quasiperiodic and chaotic dynamics. Essential for understanding parameter sensitivity and clinical applications.

## Key Contributions

- Complete bifurcation diagram for Jansen-Rit parameters
- Identification of codimension-1 and codimension-2 bifurcations
- Routes to chaos through quasiperiodic regimes
- Parameter sensitivity analysis for clinical relevance

## Dynamical Regimes

1. **Low activity**: Stable fixed point (resting state)
2. **Alpha rhythm**: Limit cycle oscillation (8-13 Hz)
3. **Epileptic**: High-amplitude rhythmic activity
4. **Chaotic**: Irregular dynamics (possible interictal state)
