# Large-scale brain dynamics of seizures: asymptotic analysis of a neural field model

**Authors**: Michael Breakspear, John A. Roberts, John R. Terry, Stefano Rodrigues, Nader Mahmud, Philip Robinson  
**Published**: 2006  
**Journal**: Journal of Computational Neuroscience  
**DOI**: 10.1007/s10827-006-8135-2

## Summary

Extension of neural field theory to model primary generalized seizures using a spatially distributed version of the Jansen-Rit model. Combines local columnar dynamics with long-range connectivity to capture seizure propagation.

## Key Contributions

- Spatially distributed Jansen-Rit model (neural field variant)
- Asymptotic analysis reducing high-dimensional model to low-dimensional dynamics
- Connection of seizure dynamics to bifurcation topology
- Framework for large-scale seizure modeling in whole-brain context

## Clinical Relevance

Models absence seizures (3 Hz spike-wave) and tonic-clonic transitions through parameter changes that shift the system through different bifurcation regimes—directly relevant to clinical seizure understanding.
