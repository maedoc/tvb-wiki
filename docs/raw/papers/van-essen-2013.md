# The WU-Minn Human Connectome Project: An Overview

**Authors:** David C. Van Essen, Kamil Ugurbil, et al.  
**Year:** 2013  
**Venue:** NeuroImage

## Key Contributions

- Comprehensive overview of the Human Connectome Project (HCP) goals and methods
- Detailed description of multimodal neuroimaging acquisition protocols
- Standardized pipelines for structural and functional connectivity mapping
- Open data sharing framework for the neuroscience community

## Abstract Summary

The Human Connectome Project (HCP) is a major effort to map human brain connectivity in unprecedented detail. This paper provides an overview of the WU-Minn consortium's contributions, including optimized acquisition protocols for structural MRI, functional MRI, and diffusion imaging at 3T and 7T. The project aims to characterize brain connectivity in 1200 healthy young adults.

## Significance

The HCP has provided the neuroscience community with unprecedented datasets and standardized protocols for studying human brain connectivity. The openly shared data has enabled thousands of studies worldwide.
