# Functional Connectivity in the Resting Brain

**Authors:** Michael D. Greicius, Ben Krasnow, Allan L. Reiss, Vinod Menon  
**Year:** 2003  
**Venue:** PNAS

## Key Contributions

- First demonstration of functional connectivity within DMN using resting-state fMRI
- Showed correlation between posterior cingulate and medial prefrontal cortex
- Established seed-based correlation analysis for resting-state data
- Validated DMN as a functionally connected network

## Abstract Summary

Using resting-state fMRI, the authors demonstrated that brain regions identified as the default mode network show strong temporal correlations in spontaneous activity. Seed-based correlation analysis revealed functional connectivity between posterior cingulate cortex, medial prefrontal cortex, and lateral parietal cortex.

## Significance

Landmark paper establishing resting-state functional connectivity as a method for studying large-scale brain networks, validating the DMN as a coherent functional system.
