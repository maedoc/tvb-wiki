# Statistical parametric maps in functional imaging: A general linear approach

**Authors:** Karl J Friston, Andrew P Holmes, Keith J Worsley, Jean-Baptiste Poline, Chris D Frith, Richard SJ Frackowiak  
**Year:** 1994  
**Venue:** Human Brain Mapping 2(4):189-210

## Summary
Foundational paper introducing Statistical Parametric Mapping (SPM) for analyzing functional neuroimaging data. Established the general linear model (GLM) framework that became the standard for fMRI data analysis.

## Key Contributions
- General Linear Model framework for neuroimaging analysis
- Random field theory for multiple comparisons correction
- SPM software foundation

## Significance for Whole-Brain Modeling
Statistical methods for comparing model outputs to empirical data. The GLM approach influences how model predictions are tested against observed activation patterns.
