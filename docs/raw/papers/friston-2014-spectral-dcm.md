## Summary
Extends DCM to resting-state fMRI using spectral analysis. Introduces methods for inferring effective connectivity from spontaneous fluctuations without experimental perturbations.

## Key Concepts
- Spectral DCM for resting-state
- Endogenous fluctuations
- Stability and connectivity
- Model reduction techniques
- Stationary multivariate processes
