# Analyzing Neural Time Series Data: Theory and Practice

**Author:** Mike X Cohen  
**Year:** 2014  
**Venue:** MIT Press

## Summary
Comprehensive guide to analyzing EEG and MEG time series data. Covers preprocessing, time-frequency analysis, connectivity measures, and statistical testing with practical MATLAB examples.

## Key Contributions
- Time-frequency analysis methods
- Connectivity analysis (spectral, phase)
- Statistical testing for neuroimaging
- Practical implementation guidance

## Significance for Whole-Brain Modeling
Provides analysis methods for comparing model outputs to empirical EEG/MEG data. Time-frequency and connectivity measures are key validation targets for models.
