# PyNEST: A convenient interface to the NEST simulator

**Authors**: Eppler et al. (2009)
**Journal**: Frontiers in Neuroinformatics
**DOI**: https://doi.org/10.3389/neuro.11.012.2008
**Full text**: https://doi.org/10.3389/neuro.11.012.2008

## Summary

This paper introduces PyNEST, a Python interface to the NEST simulator that exposes NEST's full functionality through a high-level scripting API. PyNEST allows researchers to define neuron populations, synaptic connections, and simulation parameters using concise Python code, and integrates seamlessly with the broader scientific Python ecosystem (NumPy, Matplotlib, SciPy). The interface enables rapid model prototyping and reproducible sharing of simulation scripts. By lowering the programming barrier to NEST, PyNEST has substantially broadened NEST's user base and facilitated the development of complex simulation workflows.

## Key Contributions

- Python interface to NEST simulator
- High-level scripting API
- Integration with NumPy, Matplotlib, SciPy
- Rapid model prototyping
- Reproducible simulation script sharing
