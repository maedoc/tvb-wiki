# Collective Dynamics of 'Small-World' Networks

**Authors:** Duncan J. Watts, Steven H. Strogatz  
**Year:** 1998  
**Venue:** Nature

## Key Contributions

- Introduction of the small-world network model
- Demonstration that small-world topology balances local clustering with short path lengths
- Analysis of dynamics on small-world networks
- Application to real-world networks including neural systems

## Abstract Summary

This paper introduces the small-world network model, showing that networks with high clustering coefficients and short average path lengths exhibit special dynamical properties. The authors demonstrate that many real-world networks, including the neural network of C. elegans, have small-world topology.

## Significance

One of the most influential papers in network science. The small-world concept has been widely applied to brain networks, where it is thought to support both local processing and global integration.
