# Steven Smith

**Born:** 1968  
**Nationality:** British  
**Institution:** University of Oxford, FMRIB Centre

## Overview
Steven Smith is a British neuroimaging researcher and lead developer of FSL (FMRIB Software Library), one of the most widely used neuroimaging analysis packages. His research spans resting-state connectivity, brain networks, and neuroimaging methodology.

## Key Contributions
- Lead developer of FSL (FMRIB Software Library)
- **2009**: Demonstrated correspondence between task activation and resting-state networks
- Developed methods for ICA-based analysis of resting-state data
- Advanced understanding of intrinsic connectivity networks
- Created tools for brain extraction, registration, and statistical analysis

## Significance
Smith's work on resting-state networks and his software tools have been fundamental to the field. The correspondence between task and rest he demonstrated supports using resting-state data to parameterize models for task simulations.
