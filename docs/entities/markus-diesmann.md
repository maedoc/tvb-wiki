# Markus Diesmann

Computational neuroscientist and lead developer of the NEST simulator.

## Research Focus

- Large-scale spiking neural network simulation
- NEST simulator development and architecture
- Cortical microcircuit modeling
- High-performance computing for neuroscience

## Key Publications

- Gewaltig & Diesmann (2007) — NEST introduction gewaltig-diesmann-2007
  - Scholarpedia article introducing NEST simulator
  - Large networks of point neurons
  - MPI and OpenMP parallel capabilities

- Potjans & Diesmann (2014) — Cortical microcircuit potjans-diesmann-2014
  - Data-driven spiking network model of cortical layers 2/3-6
  - Four cell types per layer
  - Canonical reference circuit for NEST validation
