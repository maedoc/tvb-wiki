# Structural Core

A set of highly interconnected hub regions that form a central backbone for brain communication.

## Definition

The structural core refers to a group of brain regions that are densely interconnected and occupy a central position in the brain's structural network. These regions form a "rich club" of high-degree nodes that facilitate global information integration.

## Discovery

patric hagmann|Hagmann et al. (2008) identified the structural core using diffusion spectrum imaging and graph analysis:
- Located in posterior medial and parietal cortex
- Includes precuneus, posterior cingulate, and superior parietal regions
- Forms the most densely interconnected part of the brain

## Characteristics

### Anatomical Location
- **Precuneus**: Central hub region
- **Posterior cingulate cortex**: Default mode network hub
- **Superior parietal cortex**: Association cortex
- **Isthmus cingulate**: Connects parietal and temporal regions

### Network Properties
- **High Degree**: Many connections to other regions
- **High Betweenness**: Central to network paths
- **Rich-Club**: Dense interconnections among core regions
- **High Efficiency**: Facilitates rapid information transfer

## Functional Significance

### Information Integration
- Serves as a central "backbone" for brain communication
- Enables efficient information transfer between modules
- Supports global workspace function

### Resting-State Connectivity
- Core regions show strong functional connectivity at rest
- Part of the default mode network
- Active during internally-directed cognition

### Metabolism
- High baseline metabolic activity
- High vascular density
- Part of the "default mode" of brain function

## Relationship to Other Concepts

### Rich-Club
The structural core is a manifestation of rich-club organization – high-degree nodes that are densely interconnected.

### Default Mode Network
Core regions overlap substantially with the default mode network, suggesting a structural basis for the DMN.

### Connector Hubs
Core regions function as connector hubs, linking different functional modules.

## Clinical Relevance

### Disease
- **Alzheimer's**: Core regions show early amyloid deposition
- **Disorders of Consciousness**: Core damage impairs awareness
- **Psychiatric Disorders**: Altered core connectivity

### Neuroimaging
- Used as seed regions for resting-state analysis
- Important for connectivity mapping
- Target for brain stimulation studies
