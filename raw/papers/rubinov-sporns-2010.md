# Complex Network Measures of Brain Connectivity: Uses and Interpretations

**Authors:** Mikail Rubinov, Olaf Sporns  
**Year:** 2010  
**Venue:** NeuroImage

## Key Contributions

- Comprehensive review of graph theory measures for brain connectivity analysis
- Introduction of the Brain Connectivity Toolbox (BCT)
- Guidelines for interpreting network measures in neuroscience contexts
- Discussion of weighted vs binary networks, directed vs undirected graphs

## Abstract Summary

This paper reviews and introduces complex network measures derived from graph theory for analyzing brain connectivity data. The authors discuss the interpretation of these measures in neuroscience contexts and introduce the Brain Connectivity Toolbox, a widely-used software package for network analysis.

## Significance

The Brain Connectivity Toolbox (BCT) has become one of the most widely used software packages for brain network analysis. This paper serves as both a methodological guide and a reference for interpreting network measures.
