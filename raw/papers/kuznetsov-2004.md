## Summary
Comprehensive reference on applied bifurcation theory covering numerical methods for continuation, normal form analysis, and software tools (MATCONT, AUTO). Essential for computational analysis of dynamical systems.

## Key Concepts
- Numerical continuation methods
- Detection and analysis of bifurcation points
- Normal form computation
- Software implementation (AUTO, MATCONT)
- Codimension 1 and 2 bifurcations

## Relevance to TVB
- Methods for [[bifurcation analysis]] in neural models
- Parameter space exploration for [[epilepsy modeling]]
- Continuation tools for [[neural mass model]] calibration
