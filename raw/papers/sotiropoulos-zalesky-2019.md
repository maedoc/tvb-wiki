# Building connectomes using diffusion MRI: Why, how and but

**Authors:** Stamatos N Sotiropoulos, Andrew Zalesky  
**Year:** 2019  
**Venue:** NMR in Biomedicine 32(4):e3752

## Summary
Comprehensive review of connectome construction from diffusion MRI. Covers acquisition, preprocessing, tractography, connectivity matrix generation, and validation. Discusses open challenges and future directions.

## Key Contributions
- Connectome construction pipeline
- Best practices for diffusion MRI
- Validation and quality control
- Limitations and future directions

## Significance for Whole-Brain Modeling
Essential reference for structural connectivity data used in models. Understanding connectome construction helps interpret model inputs and assess reliability.
