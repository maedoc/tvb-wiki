## Abstract

Organoid Intelligence (OI) represents an emerging paradigm in biocomputing, leveraging the computational capabilities of lab-grown neural cultures. While promising, conceptualizing and testing the computational architectures required for stable learning remains a significant challenge. This paper presents a simulated framework to address this gap. We designed and implemented two distinct models of a biocomputing agent embodied in a simple game environment: a V1.0 model using a basic biofeedback algorithm, and an architecturally upgraded V2.0 model incorporating momentum-based learning and lateral inhibition. This evaluation demonstrates that the V1.0 model exhibits “fragile learning,” characterized by initial performance gains folloIt has beend by rapid decay. In contrast, the calibrated V2.0 model successfully overcomes this instability, achieving a significantly higher and more stable performance plateau. This work provides a tangible blueprint for an OI system and demonstrates that architectural features designed to stabilize memory and clarify signals are critical for transitioning from simple reactivity to robust, learned behavior in biocomputing agents.
