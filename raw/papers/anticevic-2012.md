# The Role of the Default Network in Cognitive Control

**Authors:** Alan Anticevic, Michael W. Cole, et al.  
**Year:** 2012  
**Venue:** Neuron

## Key Contributions

- Demonstrated DMN deactivation is critical for cognitive control
- Showed DMN suppression facilitates task performance
- Used computational modeling to understand DMN-task network interactions
- Found impaired DMN suppression in schizophrenia

## Abstract Summary

The authors investigated the relationship between default mode network activity and cognitive control. They found that successful task performance requires suppression of DMN activity, and that this suppression is impaired in schizophrenia. Computational modeling suggested DMN and task-positive networks are anticorrelated.

## Significance

Important work establishing the functional significance of DMN deactivation for cognition, with clinical implications for understanding psychiatric conditions.
