# Independent component analysis of electroencephalographic data

**Authors:** Scott Makeig, Anthony J Bell, Tzyy-Ping Jung, Terrence J Sejnowski  
**Year:** 1996  
**Venue:** NIPS 8:145-151

## Summary
Introduced Independent Component Analysis (ICA) as a method for separating mixed EEG signals into functionally distinct sources. Enabled artifact removal and identification of brain-derived components.

## Key Contributions
- ICA application to EEG/MEG data
- Blind source separation for neuroimaging
- Artifact removal (eye movements, muscle activity)
- Component-based analysis approach

## Significance for Whole-Brain Modeling
ICA reveals underlying network components that models should capture. Component time courses provide targets for validating model-generated activity patterns.
