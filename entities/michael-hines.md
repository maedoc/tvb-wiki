# Michael Hines

Lead developer of the NEURON simulation environment for multi-compartment neuron modeling.

## Research Focus

- Multi-compartment neuron simulation
- NEURON simulator development
- Numerical methods for neural differential equations
- Biophysically detailed neuron modeling

## Key Publications

- Hines & Carnevale (1997) — The NEURON simulation environment hines-carnevale-1997
  - NEURON introduction and architecture
  - NMODL language for ion channel kinetics
  - Implicit numerical methods for stiff equations

- Carnevale & Hines (2006) — The NEURON Book carnevale-hines-2006
  - Comprehensive NEURON reference and tutorial
  - hoc and Python interfaces
  - Parallel network simulation
